# AI PPT 演讲视频生成器 1.0

封版日期：2026-06-24

## 版本定位

1.0 是本地单机最小可用版本，用 Streamlit 提供页面入口，将 PPT 解析、AI 讲稿生成、语音生成、页面导图和 MP4 合成串成一个完整流程。

## 已包含功能

- 上传 `.pptx` 文件并解析每页标题和文本内容
- 按目标总时长和讲稿风格生成逐页讲稿
- 支持 Edge、MiniMax、Windows 本机语音引擎
- 导出 PPT 页面图片
- 使用 ffmpeg 合成单页视频并拼接最终 MP4
- 调试模式支持只跑前 1 页，方便快速验证
- 项目级 Streamlit 浅色主题配置

## 暂不包含

- 云端账号系统
- 在线协作
- 数字人形象
- 模板市场
- 生产级任务队列和部署
