# AI PPT 演讲视频生成器 1.2

封版日期：2026-06-25

## 版本定位

1.2 是任务过程可见化版本。重点不是新增复杂能力，而是让“上传 PPT -> 讲稿 -> 语音 -> 图片 -> 视频”的长流程更清楚、更可控、更适合演示。

## 已包含功能

- 讲稿预览流程显示解析和 LLM 生成状态
- 一键生成视频流程显示阶段状态：
  - 解析 PPT / 生成讲稿
  - 生成语音
  - 导出页面图片
  - 合成视频片段
  - 拼接最终视频
  - 保存最终视频
- 合成视频片段时显示当前页数
- 失败时提示当前阶段，例如“生成语音失败”“导出页面图片失败”
- 视频生成完成后显示实际时长和目标时长对比
- 支持下载中间结果：
  - `slides.json`
  - `scripts.json`
  - `scripts_edited.json`
  - `audio.zip`
  - `images.zip`
  - `final.mp4`
- 增加 `pytest.ini`，确保 `pytest -q` 只收集 `tests/` 目录
- 修复 `ffprobe` 路径查找问题，避免读取视频时长阶段误替换目录名

## 暂不包含

- 后台任务队列
- 浏览器关闭后继续运行
- 多任务并发
- 任务历史记录
- 独立任务目录结构

## 面试表达

1.2 版本主要解决长流程 AI 应用的可观测性问题。视频生成不是一个瞬时动作，而是由讲稿、语音、导图、片段合成和最终拼接组成的多阶段流水线。我把这些阶段展示在页面上，并在失败时提示具体阶段，让用户知道系统正在做什么，也方便开发者定位问题。
